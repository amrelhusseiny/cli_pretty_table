# cli_pretty_table
 converts a 2d dictionary to a CLI pretty table 
## Options : 
    --json_raw converts json list to text table 
        Example : 
            python cli_pretty_table.py --json_raw '[{"name":"amr","age":"20"}]'
            +------+-----+
            | name | age |
            +------+-----+
            | amr  | 20  |
            +------+-----+
    --json_file (not operational yet)
